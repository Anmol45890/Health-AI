# 🏥 Health-AI: Multi-Disease Prediction System using Machine Learning

## 📌 Overview
**Health-AI** is a machine learning-based healthcare application that predicts the likelihood of multiple diseases based on user-provided medical data.  
It helps in early detection, risk assessment, and decision support using AI models.

---

## 🎯 Objective
- Apply machine learning in healthcare  
- Predict diseases using patient data  
- Provide fast and accurate risk analysis  
- Demonstrate real-world ML application  

---

## 🚀 Features
- 🔍 Multi-disease prediction system:
  - Diabetes Prediction  
  - Heart Disease Prediction  
  - Breast Cancer Prediction  
  - Stroke Prediction  
  - Hypertension Prediction  

- 📊 Real-time predictions  
- 🖥️ Interactive UI using Streamlit  
- 📈 ML-based insights
